This project is a simple user management CRUD application which is developed by JAVA (version 8).
	A User has: name, email, age and a phone number
		Restriction: User age should be received from the endpoint: https://api.agify.io/?name=?

Technologies:
	Java - version 8
	Spring Boot - 2.6.0
	Maven
	MySQL
	Junit
	MockMvc

Run the app in the Development Mode.
	By using UserInformationSystemApplication.class, one can run the application. Then will automatically start the project from 8080 port.
Run the app in the Test Mode.
		By using UserInformationSystemApplicationTests.class, one can run the tests with port 8001.
